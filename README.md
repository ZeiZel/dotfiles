## For working zsh

Create a hard symlink:

```shell
 ln .zshrc ~/.zshrc
 ln .tmux ~/.tmuc.conf
```

## Installing Tmux

Enter the Tmux

```bash
tmux
```

Install plugins with `ctrl + a` + `shift + i`

## For working nvim

Run `:MasonInstall` and `:Lazy install` (all packages is ensured)
