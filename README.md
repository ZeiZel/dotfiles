# Dotfiles

## Installing

```bash
sh -c "$(curl -fsSL https://github.com/ZeiZel/dotfiles/raw/master/install.sh)"
```

### For working nvim

Run `:MasonInstall` and `:Lazy install` (all packages is ensured)

## Utils

- Ghostty
- tiling - aerospace
- history - autuin
- theme - starship
